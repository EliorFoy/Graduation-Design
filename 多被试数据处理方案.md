# 多被试数据处理方案

## 📊 为什么要用全部 9 名被试？

### 必要性分析

#### 1. **科学性与说服力**
- ❌ **单被试问题**：评委会质疑"只是个例，不具普遍性"
- ✅ **多被试优势**：证明方法在不同个体上都有效，结论更可靠

#### 2. **统计学要求**
- 需要计算**组水平（group-level）**统计指标
- 均值±标准差才有意义（如 85.8% ± 4.8%）
- 才能做显著性检验（t-test、ANOVA）

#### 3. **BCI 竞赛标准**
- BCIC IV-2a 官方评估就是 9 名被试
- 经典论文（如 FBCSP）都报告 9 人平均结果
- 便于与文献对比

#### 4. **揭示个体差异**
- 不同被试的准确率波动（70%-90%）
- 分析为什么有些人效果好、有些人差
- 这是 BCI 领域的重要研究问题

## 💻 实现方案

### 方案 1：批量处理脚本（推荐）

你已经有了 `general_process.py`，只需稍作修改：

```python
# general_process.py - 批量处理版本
from pretreatment.eeg_analysis import get_modified_raw_data
from pretreatment.complete_preprocessing import complete_preprocessing_pipeline
from feature_extraction.csp_feature import extract_csp_features
from feature_extraction.wavelet_feature import extract_wavelet_energy_features, normalize_features
from classification.svm_classifier import train_svm_classifier, evaluate_classifier
import numpy as np
import pandas as pd

# 定义所有被试
subjects = ['A01T', 'A02T', 'A03T', 'A04T', 
            'A05T', 'A06T', 'A07T', 'A08T', 'A09T']

# 存储结果
results = []

for subject_id in subjects:
    print(f"\n{'='*60}")
    print(f"处理被试：{subject_id}")
    print(f"{'='*60}\n")
    
    try:
        # Step 1: 预处理
        epochs = complete_preprocessing_pipeline(subject=subject_id)
        
        # Step 2: 特征提取
        X_csp, csp_model = extract_csp_features(epochs, n_components=4)
        X_wavelet = extract_wavelet_energy_features(epochs, wavelet='db4', level=4)
        
        # Step 3: 特征融合
        X_csp_norm, scaler_csp = normalize_features(X_csp)
        X_wavelet_norm, scaler_wavelet = normalize_features(X_wavelet)
        X_fused = np.hstack([X_csp_norm, X_wavelet_norm])
        
        # Step 4: 获取标签
        y = np.array(epochs.events[:, 2], dtype=int)
        
        # Step 5: 10 折交叉验证
        from sklearn.model_selection import cross_val_score, StratifiedKFold
        from sklearn.svm import SVC
        
        cv = StratifiedKFold(n_splits=10, shuffle=True, random_state=42)
        clf = SVC(kernel='rbf')
        
        scores = cross_val_score(clf, X_fused, y, cv=cv, scoring='accuracy')
        
        # 记录结果
        results.append({
            'subject': subject_id,
            'accuracy_mean': scores.mean(),
            'accuracy_std': scores.std(),
            'n_trials': len(y)
        })
        
        print(f"{subject_id} 完成：{scores.mean()*100:.2f}% ± {scores.std()*100:.2f}%")
        
    except Exception as e:
        print(f"❌ {subject_id} 处理失败：{e}")
        results.append({
            'subject': subject_id,
            'accuracy_mean': None,
            'accuracy_std': None,
            'n_trials': None,
            'error': str(e)
        })

# 保存结果
df_results = pd.DataFrame(results)
print("\n" + "="*60)
print("所有被试处理完成！")
print("="*60)
print(df_results)

# 保存到 CSV
df_results.to_csv('all_subjects_results.csv', index=False)
print("\n结果已保存到：all_subjects_results.csv")

# 计算组水平统计
valid_results = df_results.dropna(subset=['accuracy_mean'])
if len(valid_results) > 0:
    group_mean = valid_results['accuracy_mean'].mean()
    group_std = valid_results['accuracy_mean'].std()
    print(f"\n组水平准确率：{group_mean*100:.2f}% ± {group_std*100:.2f}%")
```

## ❓ 常见问题

### Q1：如果某个被试效果特别差怎么办？
**A**：
- 如实报告，不要隐藏
- 分析原因（伪迹多？注意力不集中？）
- 这正好是"个体差异"的例证
- 可以在讨论部分深入分析

### Q2：是否需要剔除效果差的被试？
**A**：
- ❌ **不要随意剔除**（除非有明显伪迹）
- ✅ 保留所有数据，体现实事求是的科学态度
- ✅ 分析为什么这个被试效果差（可能是重要发现）

### Q3：9 个人结果差异很大怎么办？
**A**：
- ✅ 这是正常现象！BCI 领域普遍存在个体差异
- ✅ 计算平均值±标准差
- ✅ 讨论可能的原因（解剖结构、心理状态、疲劳程度）
- ✅ 提出改进方向（个性化校准、迁移学习）
