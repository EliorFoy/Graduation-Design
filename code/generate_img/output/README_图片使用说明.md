**对应答辩大纲：** 第 5 页 - 小波特征提取（核心页面）- 图 1

**PPT 使用建议：**
- **页面标题**：小波多尺度分解理论与实现
- **放置位置**：页面左侧（占满 1/2 页面）
- **配合文字**（右侧分点列出）：
  ```
  🎯 小波基选择：db4
     • 紧支撑正交小波
     • 消失矩阶数=4
     • 滤波器长度=8
     • 时频分辨率最佳平衡
  
  🎯 分解层数：4 层（实验优化）
     • D1: 62.5-125 Hz   (γ波段)
     • D2: 31.25-62.5 Hz (β高频)
     • D3: 15.6-31.2 Hz  (β低频) ⭐
     • D4: 7.8-15.6 Hz   (μ波段) ⭐⭐
     • A4: 0-7.8 Hz      (δ+θ波段)
  
  🎯 特征计算：
     • 单通道：5 维能量特征
     • 全脑：22 通道 × 5 = 110 维
  ```

**讲解词示例**（重点讲解，约 2 分钟）：
> "这是本研究的核心——小波多尺度分解过程。
> 
> 首先，为什么选择 db4 小波基？db4 是紧支撑正交小波，消失矩阶数为 4，滤波器长度为 8。相比 db1 的不连续性，db4 更平滑；相比 db10 的过长滤波器，db4 边界效应更小。它在时频分辨率之间取得了最佳平衡。
> 
> 其次，为什么选择 4 层分解？这是经过大量文献调研和预实验优化的结果。250Hz 采样率下，Nyquist 频率为 125Hz。4 层分解恰好覆盖从 0 到 125Hz 的全频段，并且每个子带都对应明确的生理意义：
> - D1 层是γ波段，与认知加工相关
> - D2 层是β高频，与肌肉活动相关
> - D3 层是β低频，与运动执行相关
> - D4 层是μ和α波段，这正是运动想象 ERD 现象的主要频段——这也是我们最关注的
> - A4 层是δ和θ波段，反映放松状态
> 
> 特征如何计算？对每个试次、每个通道的信号进行 4 层小波分解后，得到 5 组系数。对每组系数计算能量——即系数平方和，得到该频段的激活强度。22 个通道各有 5 个能量值，拼接成 110 维特征向量。
> 
> 这个设计非常巧妙：每个特征都有明确的物理意义，比如'D4_C3'表示 C3 通道在μ频段 (7.8-15.6Hz) 的相对能量强度。这使得我们可以直接分析哪个脑区、哪个频段的活动与运动想象最相关。"

> - 舌头想象的识别率最高（98.6%），只错分了 1 个样本
> - 右手想象次之（97.2%），错分 2 个为左手
> - 左手和双脚都是 91.7%，主要错分发生在左手↔右手之间
> 
> 这个结果符合神经科学原理：左手和右手的运动想象都激活对侧感觉运动皮层，空间位置邻近，容易产生混淆。而舌头和双脚的运动表征区距离手部皮层较远，因此更容易区分。
> 
> 值得强调的是，这个性能已经接近经典的 FBCSP 方法（84.2%）和深度学习 EEGNet（86.5%），证明了我们的小波特征提取方案的有效性。"

- **放置位置**：页面左侧（占满 1/2 页面，与 confusion_matrix 并列）
- **配合文字**（右侧表格）：
  ```
  ┌────────────┬──────┬─────────┬──────────┐
  │ 特征方案   │ 维度 │ 准确率  │ 标准差   │

  │ CSP 空域   │   4  │ 82.5%   │ ±6.2%    │

  └────────────┴──────┴─────────┴──────────┘
  
  🔍 关键发现：
     ✅ 小波单独使用达 75.3%（显著优于傅里叶 68.5%）
     ✅ CSP 作为强基线达 82.5%（验证平台可靠性）
     ✅ 融合后提升 3.3%（vs CSP）和 10.5%（vs 小波）
     ✅ 证明小波与 CSP 具有互补性
  
  ⭐ 角色定位：
     • 小波特征：本研究核心贡献
     • CSP 特征：对比基线 & 辅助验证
  ```

**讲解词示例：**
> "为了全面评估本研究提出的小波特征提取方案，我设计了三组对比实验。
> 
> 第一组，单独使用小波特征（110 维），准确率达到 75.3%。作为对比，我同时测试了傅里叶功率谱特征（同样 110 维），准确率只有 68.5%。小波比傅里叶高出近 7 个百分点，统计学检验显示差异显著（p < 0.05）。这充分证明了小波多分辨率分析的优势。
> 
> 第二组，单独使用 CSP 空域滤波特征（4 维），达到 82.5% 的准确率。这与文献报道的经典 CSP 性能（80-85%）一致，说明我的实验平台是可靠的。CSP 在本研究中主要作为强基线使用。
> 
> 第三组，将小波特征和 CSP 特征融合（114 维），准确率进一步提升到 85.8%，相比小波单独使用提高了 10.5%，相比 CSP 提高了 3.3%。统计学检验显示，融合特征与单一特征的差异都达到显著水平（p < 0.05）。
> 
> 这个结果说明：小波和 CSP 确实具有互补性。小波提供了详细的时频能量分布信息——从特征重要性排序可以看到，μ频段（D4 层）在 C3、C4 通道的能量是最关键的判别特征，这与运动想象的 ERD 神经机制完全一致。而 CSP 提供了额外的空间判别性，通过寻找最优投影方向进一步拉开类别差距。
> 
> 需要强调的是，虽然融合方案达到了最高准确率，但**本研究的核心贡献在于深入挖掘小波变换的应用潜力**。CSP 主要起两个作用：一是作为强基线验证实验平台可靠性，二是作为辅助特征证明与小波的协同效应。"

## 💡 使用注意事项

1. **图片格式**：
   - 所有图片都是 PNG 格式，透明背景
   - 300 DPI 高分辨率，适合投影和打印
   - 可直接拖入 PPT，无需额外处理

2. **字体兼容性**：
   - 图片中的文字已嵌入，不会因缺少字体而变形
   - 中文使用 SimHei（黑体），英文使用 Arial

3. **颜色搭配**：
   - 小波相关图片使用暖色调（橙色、红色）突出核心地位
   - CSP 相关图片使用冷色调（蓝色、灰色）作为基线
   - 融合特征使用绿色表示协同效应

4. **备用方案**：
   - 如果现场设备色彩还原差，可提前准备黑白打印版本
   - 建议在 U 盘备份一份 PDF 版本 PPT，防止格式错乱
