# BCIC IV-2a 数据集详解

## 📖 数据集概述

**BCI Competition IV Dataset 2a** 是脑机接口（Brain-Computer Interface, BCI）领域最常用的公开数据集之一，由奥地利格拉茨理工大学（Graz University of Technology）提供。

### 基本信息

| 属性 | 值 |
|------|-----|
| **数据来源** | BCI Competition IV (2008) |
| **实验类型** | 运动想象（Motor Imagery, MI） |
| **被试数量** | 9 人（A01-A09） |
| **通道数量** | 22 个 EEG 通道 + 3 个 EOG 通道 |
| **采样率** | 250 Hz |
| **信号带宽** | 0.5-100 Hz（已陷波滤波去除 50Hz 工频干扰） |
| **任务类别** | 4 类运动想象 |
| **文件类型** | GDF (General Data Format) |

---

## 🧠 实验任务设计

### 4 类运动想象任务

被试需要想象以下 4 种肢体运动：

| 类别代码 | 想象动作 | 对应大脑区域 |
|---------|---------|-------------|
| **1** | 左手运动想象 (Left Hand) | 右侧运动皮层 (C4 附近) |
| **2** | 右手运动想象 (Right Hand) | 左侧运动皮层 (C3 附近) |
| **3** | 双脚运动想象 (Both Feet) | 顶部运动皮层 (Cz 附近) |
| **4** | 舌头运动想象 (Tongue) | 底部运动皮层 (FCz 附近) |

### 实验范式

```
时间线：
├─ 0.0s - 2.0s:  静息状态（注视屏幕中央"+"）
├─ 2.0s:        提示音（蜂鸣声）
├─ 2.0s - 3.25s: 屏幕显示箭头提示（← → ↓ ↑）
├─ 3.25s - 6.0s: 被试执行运动想象任务
└─ 6.0s - 7.0s:  休息间隔

单次试次（Trial）时长：约 7 秒
```

## 📊 数据格式详解

### GDF 文件格式

**GDF (General Data Format)** 是生物医学信号的标准格式，包含：

1. **数据部分**
   - 多通道时间序列数据
   - 单位：微伏 (μV)

2. **元数据（Header）**
   - 采样率：250 Hz
   - 通道数：25（22 EEG + 3 EOG）
   - 数据类型：float32
   - 参考电极：原始参考（通常是乳突或鼻尖）

3. **标注（Annotations）**
   - 事件标记（Event markers）
   - 试次起始位置
   - 任务类别标签

### 数据维度示例

```python
# 加载后的数据形状
data.shape = (n_channels, n_times)
           = (25, 约 600,000)  # 取决于记录时长

# 单次试次（4 秒任务段）
epoch.shape = (22, 1000)  # 22 通道 × 1000 时间点 (4 秒 × 250Hz)
```

### 时间分辨率计算

```
采样率：250 Hz
→ 每个采样点时间间隔：1/250 = 0.004 秒 = 4 毫秒
→ 1 秒数据：250 个采样点
→ 4 秒试次：1000 个采样点
```

## 🎯 数据特征与物理意义

### 1. 运动想象产生的脑电信号

**感觉运动节律（Sensorimotor Rhythms, SMR）**

| 频段 | 频率范围 | 生理意义 | 与运动想象的关系 |
|------|---------|---------|----------------|
| **μ节律** | 8-13 Hz | 感觉运动皮层静息节律 | 运动想象时**抑制**（ERD） |
| **β节律** | 13-30 Hz | 运动维持和反馈 | 运动想象时先抑制后增强（ERD/ERS） |
| **γ节律** | 30-100 Hz | 高级认知处理 | 注意力集中时增强 |

### 2. 事件相关去同步/同步（ERD/ERS）

```
功率变化
  ↑
  |         ERS (增强)
  |        ╱
  |───────╱────────── 基线
  |      ╲
  |       ╲ ERD (抑制)
  |        ╲
  ↓
  ←─── 时间 ───→
     想象开始
```

- **ERD (Event-Related Desynchronization)**: 功率下降，皮层激活
- **ERS (Event-Related Synchronization)**: 功率上升，皮层抑制

### 3. 空间分布特征

```
左手想象 → 右侧运动皮层激活 (C4 区域 μ节律抑制)
右手想象 → 左侧运动皮层激活 (C3 区域 μ节律抑制)
双脚想象 → 顶部运动皮层激活 (Cz 区域 μ节律抑制)
舌头想象 → 底部运动皮层激活 (FCz 区域 μ节律抑制)
```

## 🔧 预处理参数选择的物理依据

### 1. 滤波设置（0.5-40Hz）

| 滤波器 | 参数 | 物理意义 |
|--------|------|---------|
| **高通** | 0.5 Hz | 去除基线漂移和缓慢电位变化 |
| **低通** | 40 Hz | 保留μ和β节律，去除高频噪声 |
| **陷波** | 50 Hz | 去除工频干扰（欧洲标准） |

### 2. 为什么选择这个频段？

```
频段分析：
├─ δ (0.5-4 Hz):   睡眠波，通常去除
├─ θ (4-8 Hz):     困倦/记忆，保留部分
├─ α/μ (8-13 Hz):  【核心】感觉运动节律，必须保留
├─ β (13-30 Hz):   【核心】运动相关节律，必须保留
└─ γ (30-100 Hz):  高级认知，可选（你的项目滤除）
```

### 3. ICA 去除的伪迹

| 伪迹类型 | 频率特征 | 空间分布 | ICA 成分特征 |
|---------|---------|---------|-------------|
| **EOG (眼电)** | 0.5-10 Hz | 前额通道 (Fz, FCz) | 前部权重高 |
| **EMG (肌电)** | 30-100 Hz | 全头随机 | 高频能量强 |
| **ECG (心电)** | 1-2 Hz | 全头周期性 | 周期性尖峰 |
