# 陷波滤波器（Notch Filter）详解

## 一、核心定义

陷波滤波器是一种**窄带带阻滤波器**，用于**选择性衰减特定频率及其极窄邻域**的信号成分，同时保留其他频率。在生物电信号处理中，最典型的应用是去除**工频干扰**（50 Hz 或 60 Hz）。

---

## 二、数学原理

### 1. 传递函数（连续时间域）

二阶陷波滤波器的标准传递函数为：

$$
H(s) = \frac{s^2 + \omega_0^2}{s^2 + \frac{\omega_0}{Q}s + \omega_0^2}
$$

其中：
- $\omega_0 = 2\pi f_0$：陷波中心角频率（$f_0$ 为目标滤除频率，如 50 Hz）
- $Q$：品质因数（Quality Factor），控制阻带宽度，$Q = \frac{f_0}{\Delta f}$，$\Delta f$ 为 -3 dB 带宽

### 2. 频率响应特性

令 $s = j\omega$，可得幅频响应：

$$
|H(j\omega)| = \left| \frac{-\omega^2 + \omega_0^2}{-\omega^2 + j\frac{\omega_0}{Q}\omega + \omega_0^2} \right|
$$

**关键特性**：
- 当 $\omega = \omega_0$ 时，$|H(j\omega_0)| = 0$（理论上完全衰减）
- 当 $\omega \ll \omega_0$ 或 $\omega \gg \omega_0$ 时，$|H(j\omega)| \approx 1$（信号无失真通过）
- $Q$ 越大，阻带越窄，过渡带越陡峭

**幅频响应示意图**：

```
|H(f)|
  1.0 |━━━━━━━━━━━━━━┓      ┏━━━━━━━━━━━━━━
      |              ┃      ┃
      |              ┃      ┃
  0.5 |              ┃      ┃
      |              ┃      ┃
  0.0 |--------------┻------┻--------------► f
                    f₀-Δf  f₀  f₀+Δf
                    ← 阻带宽度 →
```

## 三、离散时间实现（数字信号处理）

### 1. 双二阶（Biquad）结构

在数字系统中（如 EEG 采样后处理），常用**双二阶结构**实现。设采样频率为 $f_s$，定义：

$$
\omega_0 = 2\pi \frac{f_0}{f_s}, \quad \alpha = \frac{\sin\omega_0}{2Q}
$$

### 2. 差分方程系数

$$
\begin{aligned}
b_0 &= 1, \quad b_1 = -2\cos\omega_0, \quad b_2 = 1 \\
a_0 &= 1 + \alpha, \quad a_1 = -2\cos\omega_0, \quad a_2 = 1 - \alpha
\end{aligned}
$$

### 3. 时域差分方程

归一化后，输出 $y[n]$ 与输入 $x[n]$ 的关系：

$$
y[n] = \frac{b_0}{a_0}x[n] + \frac{b_1}{a_0}x[n-1] + \frac{b_2}{a_0}x[n-2] - \frac{a_1}{a_0}y[n-1] - \frac{a_2}{a_0}y[n-2]
$$

> ✅ 该结构计算高效、数值稳定，广泛用于嵌入式系统和实时处理。

## 四、关键参数设计指南

| 参数 | 物理意义 | 典型取值（工频抑制） | 选择建议 |
|------|----------|---------------------|----------|
| **$f_0$** | 陷波中心频率 | 50 Hz（中国/欧洲）或 60 Hz（美洲） | 根据供电标准确定 |
| **$Q$** | 品质因数 | 30–60 | $Q$ 过大→相位失真；$Q$ 过小→滤除有用信号 |
| **$\Delta f$** | -3 dB 带宽 | $\Delta f = f_0/Q \approx 1\text{–}2\,\text{Hz}$ | 需覆盖工频漂移范围（±0.5 Hz） |

### 参数选择原则

1. **中心频率 $f_0$**：
   - 中国、欧洲、亚洲大部分：50 Hz
   - 美洲、日本、韩国：60 Hz
   - 某些地区可能波动：49.5–50.5 Hz

2. **品质因数 $Q$**：
   - $Q = 30$：阻带较宽（$\Delta f \approx 1.67$ Hz），适合工频漂移大的环境
   - $Q = 50$：阻带适中（$\Delta f \approx 1$ Hz），推荐默认值
   - $Q = 60$：阻带很窄（$\Delta f \approx 0.83$ Hz），相位失真风险增加

3. **带宽计算示例**（$f_0 = 50$ Hz）：
   $$
   \Delta f = \frac{f_0}{Q} = \frac{50}{50} = 1\,\text{Hz}
   $$
   阻带范围：$[49.5, 50.5]$ Hz

## 五、应用场景

### 1. 生物电信号处理 ✅

**EEG/ECG/EMG 中去除 50/60 Hz 电源线干扰**

- **EEG**：工频干扰是主要噪声源，尤其在屏蔽不良的环境
- **ECG**：心电图信号幅度小（mV 级），易受工频影响
- **EMG**：肌电信号高频成分多，需小心选择 $Q$ 值避免失真

### 2. 音频处理

**消除特定频率的啸叫或哼声**

- 电源哼声（50/60 Hz 及其谐波）
- 声学反馈啸叫（特定谐振频率）
- 机械共振噪声

### 3. 传感器信号

**抑制机械谐振频率干扰**

- 加速度计、陀螺仪的机械谐振
- 压力传感器的固有频率
- 应变片的振动干扰

## 六、使用注意事项

### 1. 相位失真 ⚠️

**问题**：陷波滤波器是非线性相位系统，可能引起时域波形畸变。

**数学表达**：
$$
\phi(\omega) = \arg\left(H(e^{j\omega})\right) \neq \text{常数} \times \omega
$$

**对策**：采用零相位滤波（如 `filtfilt` 双向滤波）：

$$
y_{\text{zero-phase}}[n] = \text{filter}\big(\text{filter}(x[n])\text{ reversed}\big)
$$

**原理**：
- 正向滤波：$y_1[n] = h[n] * x[n]$
- 反向滤波：$y_2[n] = h[-n] * y_1[-n]$
- 总响应：$H_{\text{total}}(e^{j\omega}) = |H(e^{j\omega})|^2$，相位为零

### 2. 谐波干扰 ⚠️

**问题**：工频干扰常含 2 次、3 次谐波（100/150 Hz 或 120/180 Hz）。

**对策**：
- **方案 A**：级联多个陷波器
  $$
  H_{\text{total}}(s) = H_{50}(s) \cdot H_{100}(s) \cdot H_{150}(s)
  $$
- **方案 B**：改用**自适应陷波**（Adaptive Notch）
- **方案 C**：使用陷波 + 低通组合（如低通截止 45 Hz）

### 3. 信号频谱重叠风险 ⚠️

**问题**：若目标信号本身含 $f_0$ 附近成分（如某些神经振荡），直接陷波会损失信息。

**对策**：

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **优化硬件** | 优先优化屏蔽与接地 | 所有实验（应首选） |
| **谱插值法** | 在频域插值替换 $f_0$ 成分 | 离线分析，信号含 $f_0$ |
| **回归法** | 用参考通道回归估计工频 | 有独立噪声参考通道 |
| **时频分析** | 用小波/STFT 分离干扰 | 非平稳信号 |

---

## 七、与其他滤波器对比

| 滤波器类型 | 阻带特性 | 相位特性 | 适用场景 |
|-----------|----------|----------|----------|
| **陷波滤波器** | 极窄带阻（~1 Hz） | 非线性 | 精确去除单频干扰 |
| **带阻滤波器** | 宽带阻（>10 Hz） | 可设计为线性 | 去除频段噪声（如肌电） |
| **自适应滤波器** | 动态跟踪频率 | 依赖算法 | 工频漂移或变频干扰 |
| **硬件屏蔽 + 差分放大** | 从源头抑制 | 无数字失真 | **首选方案**，应优先实施 |

### 选择建议

1. **第一步**：优先实施硬件优化（屏蔽室、差分电极、良好接地）
2. **第二步**：如果仍有残留工频，使用数字陷波滤波器
3. **第三步**：如果工频漂移严重，考虑自适应陷波
4. **第四步**：如果信号含 $f_0$ 成分，改用谱插值或回归法

## 八、Python 实现示例

### 1. 基础实现

```python
import numpy as np
from scipy.signal import iirnotch, filtfilt

def apply_notch_filter(signal, fs, f0=50, Q=35):
    """
    对信号应用 50Hz 陷波滤波（零相位）
    
    Args:
        signal: 输入信号 (n_samples,)
        fs: 采样频率 (Hz)
        f0: 陷波中心频率 (Hz)，默认 50
        Q: 品质因数，默认 35
    
    Returns:
        filtered_signal: 滤波后信号
    """
    # 计算归一化频率（Nyquist 频率为基准）
    nyq = fs / 2.0
    f0_normalized = f0 / nyq
    
    # 设计陷波滤波器
    b, a = iirnotch(f0_normalized, Q)
    
    # 应用零相位滤波（双向滤波）
    filtered_signal = filtfilt(b, a, signal)
    
    return filtered_signal
```

### 2. 多通道 EEG 处理

```python
def apply_notch_to_eeg(epochs, f0=50, Q=35):
    """
    对多通道 EEG 数据应用陷波滤波
    
    Args:
        epochs: MNE Epochs 对象 (n_trials, n_channels, n_times)
        f0: 陷波频率
        Q: 品质因数
    
    Returns:
        epochs_filtered: 滤波后的 Epochs
    """
    data = epochs.get_data()
    fs = epochs.info['sfreq']
    n_trials, n_channels, n_times = data.shape
    
    # 逐通道滤波
    for ch_idx in range(n_channels):
        for trial_idx in range(n_trials):
            data[trial_idx, ch_idx, :] = apply_notch_filter(
                data[trial_idx, ch_idx, :], 
                fs, f0, Q
            )
    
    # 创建新的 Epochs 对象
    epochs_filtered = epochs.copy()
    epochs_filtered._data = data
    
    return epochs_filtered
```

### 3. 频响特性可视化

```python
from scipy.signal import freqz
import matplotlib.pyplot as plt

def plot_notch_response(f0=50, Q=35, fs=250):
    """绘制陷波滤波器的频响特性"""
    nyq = fs / 2.0
    b, a = iirnotch(f0 / nyq, Q)
    
    # 计算频率响应
    w, h = freqz(b, a, worN=2000)
    
    # 转换为实际频率和幅度
    freq = w * fs / (2 * np.pi)
    magnitude = 20 * np.log10(np.abs(h) + 1e-10)
    
    # 绘图
    plt.figure(figsize=(10, 4))
    plt.plot(freq, magnitude, 'b-', linewidth=2)
    plt.axvline(f0, color='r', linestyle='--', label=f'{f0} Hz')
    plt.axvspan(f0 - f0/Q, f0 + f0/Q, alpha=0.2, color='r', label=f'阻带 (Q={Q})')
    plt.xlim(0, fs/2)
    plt.ylim(-60, 5)
    plt.xlabel('频率 (Hz)')
    plt.ylabel('幅度 (dB)')
    plt.title(f'陷波滤波器频响特性 (f₀={f0}Hz, Q={Q})')
    plt.grid(True, alpha=0.3)
    plt.legend()
    plt.tight_layout()
    plt.savefig('notch_filter_response.png', dpi=300)
    print(f"✅ 频响图已保存：notch_filter_response.png")
    plt.show()

# 使用示例
plot_notch_response(f0=50, Q=35, fs=250)
```

### 4. 功率谱验证

```python
from scipy.signal import welch

def verify_notch_filter(signal, fs, f0=50, Q=35):
    """验证陷波滤波效果"""
    # 滤波
    filtered = apply_notch_filter(signal, fs, f0, Q)
    
    # 计算 PSD
    f_orig, psd_orig = welch(signal, fs, nperseg=fs*2)
    f_filt, psd_filt = welch(filtered, fs, nperseg=fs*2)
    
    # 绘图
    plt.figure(figsize=(10, 4))
    plt.semilogy(f_orig, psd_orig, label='原始信号', alpha=0.7)
    plt.semilogy(f_filt, psd_filt, label='滤波后信号', alpha=0.7)
    plt.axvline(f0, color='r', linestyle='--', label=f'{f0} Hz')
    plt.xlabel('频率 (Hz)')
    plt.ylabel('功率谱密度')
    plt.title('陷波滤波效果验证')
    plt.grid(True, alpha=0.3)
    plt.legend()
    plt.xlim(0, fs/2)
    plt.tight_layout()
    plt.savefig('notch_verification.png', dpi=300)
    print(f"✅ 验证图已保存：notch_verification.png")
    plt.show()
    
    # 计算抑制比
    idx_f0 = np.argmin(np.abs(f_filt - f0))
    suppression = 10 * np.log10(psd_orig[idx_f0] / psd_filt[idx_f0])
    print(f"📊 {f0} Hz 抑制比：{suppression:.2f} dB")
```

> 📌 **建议**：在应用陷波前，先通过**功率谱密度（PSD）分析**确认干扰频率，避免盲目滤波。

## 十、常见问题与解决方案

### 1. 工频频率不确定

**问题**：实际工频可能在 49.5–50.5 Hz 之间波动。

**解决方案**：
- 使用较宽的阻带（$Q = 30$，$\Delta f \approx 1.67$ Hz）
- 使用自适应陷波滤波器（跟踪频率漂移）
- 先用 PSD 精确估计实际工频：
  ```python
  from scipy.signal import welch
  f, psd = welch(signal, fs)
  f_50hz = f[np.argmax(psd[(f > 45) & (f < 55)])]
  print(f"实际工频：{f_50hz:.2f} Hz")
  ```

### 2. 滤波后信号失真

**问题**：时域波形出现振铃或相位偏移。

**解决方案**：
- 使用 `filtfilt` 进行零相位滤波
- 降低 $Q$ 值（如从 50 降到 30）
- 改用 FIR 陷波滤波器（线性相位）：
  ```python
  b, a = iirnotch(f0/(fs/2), Q)
  # 改为 FIR
  from scipy.signal import firwin
  b_fir = firwin(numtaps=101, cutoff=[f0-1, f0+1], fs=fs, pass_zero=False)
  ```

### 3. 多通道不一致

**问题**：不同通道的工频干扰强度不同。

**解决方案**：
- 逐通道应用陷波（不要跨通道滤波）
- 对干扰强的通道单独调整 $Q$ 值
- 考虑使用独立成分分析（ICA）分离工频

## 十一、总结

### 核心要点

| 概念 | 关键信息 |
|------|---------|
| **定义** | 窄带带阻滤波器，选择性衰减特定频率 |
| **核心参数** | $f_0$（中心频率）、$Q$（品质因数）、$\Delta f$（带宽） |
| **数学原理** | 二阶 IIR 系统，$H(s) = \frac{s^2 + \omega_0^2}{s^2 + \frac{\omega_0}{Q}s + \omega_0^2}$ |
| **数字实现** | Biquad 结构，差分方程递归计算 |
| **零相位滤波** | `filtfilt` 双向滤波消除相位失真 |
| **应用场景** | EEG/ECG/EMG 工频抑制、音频去噪、传感器信号处理 |

### 最佳实践

1. ✅ **优先硬件优化**：屏蔽室、差分电极、良好接地
2. ✅ **精确估计工频**：先用 PSD 分析确认干扰频率
3. ✅ **选择合适 $Q$ 值**：默认 $Q=35$，根据情况调整
4. ✅ **使用零相位滤波**：`filtfilt` 避免时域畸变
5. ✅ **验证滤波效果**：对比滤波前后 PSD
6. ✅ **注意谐波干扰**：必要时级联多个陷波器

### 参数推荐（EEG 工频抑制）

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| $f_0$ | 50 Hz（中国） | 根据地区调整 |
| $Q$ | 35 | 平衡阻带宽度和相位失真 |
| $\Delta f$ | ~1.4 Hz | $50/35 \approx 1.43$ Hz |
| 滤波方式 | `filtfilt` | 零相位双向滤波 |
| 谐波处理 | [50, 100, 150] | 级联去除前 3 次谐波 |
