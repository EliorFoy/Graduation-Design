## 🎯 Step 7: 重参考 (Re-reference)

### **为什么要做重参考？**

EEG 信号测量的是**相对电压差**,需要一个参考点:

```python
# 原始数据：以乳突 (Mastoid) 为参考
# 每个 EEG 通道记录的是相对于参考电极的电压
V_channel = V_scalp - V_reference
```

**问题:** 如果参考电极本身有噪声，会污染所有通道！

### **平均参考 (Average Reference)**

```python
raw_final = set_reference(raw_final, ref='average')
```

**原理:**
```python
# 计算所有 EEG 通道的平均值作为新参考
V_new = V_channel - mean(V_all_channels)

# 示例（简化到 3 个通道）:
原始值:
  C3: 10 μV
  Cz: 15 μV  
  C4: 8 μV
  
平均值 = (10 + 15 + 8) / 3 = 11 μV

重参考后:
  C3: 10 - 11 = -1 μV
  Cz: 15- 11 = 4 μV
  C4: 8 - 11 = -3 μV
```

**优势:**
- ✅ **无偏参考**: 不依赖单个电极，减少参考电极噪声
- ✅ **标准做法**: 运动想象研究的常用方法
- ✅ **提高信噪比**: 突出局部脑活动

## 🔍 常见问题解答

### Q1: 为什么 ICA 去噪后还要剔除试次？
**A:** ICA 去除的是**稳定的、可建模的**伪迹源。但瞬时伪迹（如突然的肌肉收缩）需要直接剔除整个试次。

### Q2: 1023 是谁标记的？
**A:** 数据集采集时的**实验专家**人工标记的。他们在被试实验过程中发现明显伪迹就按下标记按钮。

### Q3: 能不能不剔除，直接用所有试次？
**A:** 可以，但会**降低分类准确率**。建议对比一下:
```python
# 方案 A: 剔除伪迹试次
epochs_clean = drop_artifact_epochs(epochs, events)
accuracy_A = train_and_test(epochs_clean)  # 通常更高

# 方案 B: 保留所有试次
accuracy_B = train_and_test(epochs)  # 可能较低
```

### Q4: 还有其他剔除方法吗？
**A:** 有的！还可以基于:
- 幅度阈值（如 > 100μV 的试次）
- 方差阈值（异常波动的试次）
- 自动检测算法

## 🛠️ 分段是如何工作的？

### **代码实现**


### **内部机制**

```python
# MNE 内部逻辑（简化版）
def create_epochs(raw, tmin, tmax):
    """
    根据事件标记切割连续数据
    """
    # 1. 找到所有事件标记的位置
    events = mne.events_from_annotations(raw)
    # events.shape = (n_events, 3)
    # 每行：[时间点数，0, 事件代码]
    
    # 示例:
    events = [
        [1000, 0, 1],   # 第 1000 个采样点，左手
        [3500, 0, 2],   # 第 3500 个采样点，右手
        [6000, 0, 3],   # 第 6000 个采样点，双脚
        ...
    ]
    
    # 2. 对每个事件，截取 [tmin, tmax] 的数据
    epochs_list = []
    for event_time, _, label in events:
        start_idx = event_time + int(tmin * sfreq)
        end_idx = event_time + int(tmax * sfreq)
        
        epoch_data = raw[:, start_idx:end_idx]
        epochs_list.append(epoch_data)
    
    # 3. 堆叠成 Epochs 对象
    epochs = EpochsArray(epochs_list)
    # shape: (n_events, n_channels, n_times)
    
    return epochs
```

### **神经科学依据**

```
  0s     0.5s   1s    2s    3s    4s
```

## 📝 总结

### **分段是什么？**
> 将**连续的**EEG 信号，按照**事件标记**,切割成**离散的试次**

### **为什么要分段？**

| 原因 | 说明 |
|------|------|
| **实验对齐** | 所有试次从同一参考点（刺激出现）开始 |
| **数据组织** | 结构化存储，便于管理 |
| **标签绑定** | 每个试次绑定一个类别标签 |
| **特征提取** | 在固定时间窗内计算特征 |
| **机器学习** | 符合分类器的输入要求 |

### **类比理解**

```
分段就像剪辑电影:

原始 EEG = 整部 2 小时的电影
         ↓ 分段
Epochs = 200 个精彩片段（每个 4 秒）
         ↓ 用途
用这些片段训练 AI 识别"这是动作片还是爱情片"
```

---

现在你明白了吗？分段是连接**原始 EEG 数据**和**机器学习分类器**的关键桥梁！没有分段，就无法进行后续的特征提取和分类！😊
